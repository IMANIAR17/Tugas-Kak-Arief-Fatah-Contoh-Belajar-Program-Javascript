
# Tugas Kak Arief Fatah Contoh Belajar Program Javascript

#Soal Nomer 1
contoh program Javascript yang dapat menampilkan teks "User ke - 1 User ke - 100" pada setiap baris di halaman HTML menggunakan for loop.

#Soal Nomer 2
melakukan pengulangan menggunakan for loop untuk melakukan penambahan nilai sebanyak 10 kali, dimulai dari nilai awal 0 dengan penambahan 2 pada setiap pengulangan dan menampilkan hasil penambahan pada setiap pengulangan.

#Soal Nomer 3
melakukan pengulangan dengan for loop dari 0 sampai 20, melakukan pengecekan apakah nilai tersebut ganjil atau genap, dan menampilkan keterangan ganjil atau genap pada setiap nilai pada setiap pengulangan.

#Soal Nomer 4
menampilkan konfirmasi pop up menggunakan confirm() dengan teks "Apakah anda mau mengulang?" pada box confirm. Jika pengguna memilih "OK", maka program akan terus menampilkan pop up yang sama. Jika pengguna memilih "Cancel", maka program akan menampilkan teks "Perulangan sudah dilakukan sebanyak ... (jumlah klik OK yang dilakukan pengguna)" dan menghitung jumlah kali pengguna memilih "OK"

#Soal Nomer 5
Program Kuis untuk Menanyakan Kepanjangan dari Nama IB
Program ini menggunakan prompt() untuk meminta input dari pengguna dengan teks "Sebutkan kepanjangan dari nama IB (Impact Byte)?".

Jika jawaban pengguna benar (yaitu "Impact Byte"), maka program akan menampilkan pesan "Selamat jawaban kamu benar!" menggunakan alert().
